Aluk.js

一个集成了许多功能的Javascript Library.

Github仓库地址 前往

简介

Aluk拥有选择器功能和配套的一组函数,实现自定义元素,数据绑定,高版本JS函数向下兼容,元素扫描与批量快捷修改,扩展网页功能.

快速上手

对于Node.js(CommonJS):

const aluk = require('./aluk.min.js');

对于浏览器环境:

<script> src="./aluk.min.js"</script>

对于AMD规范:

require(['aluk.min'],(aluk) => {globalThis['aluk'] = aluk})
